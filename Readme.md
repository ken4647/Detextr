# Detextr

## basic usage

run by `python detextr.py [target_path]`



## Structure



```
| —— codes
| ———— crop_images.ipynb        # used to crop images
| ———— detextr.py
| ———— process_refs.py          # used to create new refs
| ———— run_fetgan.py            # used to run FET-GAN to transfer styles
| —— configs                    # used to save configs

```
